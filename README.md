🛡️ INFEX OSINT SYSTEM v2.0 (Elite Edition)
INFEX OSINT — это высокоэффективная портативная станция для сбора цифровой разведки (OSINT), упакованная в один автономный файл. Система предназначена для глубокого анализа цифрового следа объектов (ников, доменов, IP-адресов и почт) без необходимости использования дорогостоящих API-подписок.

💎 Особенности (Core Features)
Zero-Footprint Architecture: Система работает целиком на стороне клиента. Никаких баз данных, никакой слежки.

Deep Social Enumeration: Интеграция с крупнейшими базами (WhatsMyName, Sherlock, Social-Searcher) для деанонимизации никнеймов.

Infrastructure Analysis: Прямой доступ к Shodan, Censys и DNSDumpster для анализа сетевой структуры.

Leak & Dork Engine: Автоматизированная генерация продвинутых Google Dorks для поиска утечек, конфигов и закрытых документов.

Luxury Cyberpunk UI: Уникальный интерфейс в стиле "Golden Luxury", созданный для работы в условиях высокого давления.

🚀 Быстрый старт
Скачайте файл infex.html.

Откройте его в любом современном браузере (Chrome, Brave, Firefox).

Введите цель (Nickname, Domain или IP) в главную консоль.

Нажмите "Запустить протокол".

🛠 Технологический стек
Engine: Pure JavaScript (ES6+).

Styling: Tailwind CSS (via CDN).

Icons: FontAwesome Elite.

Frameworks: Zero (Полностью независимая сборка).
